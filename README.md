## Configuracion de envn

Para crear Entorno Virtual en python es


- En Linux

```bash
  python3 -m venv venv 
  cd venv
  cd bin 
  source activate
  deactive #para desactivar el entorno virutual
  
```
    
- En Windows

```bash
  python -m venv venv 
  cd venv
  cd Scripts 
  .\Activate.ps1
  deactive #para desactivar el entorno virutual
  
```
    
para seleccionarlo a vsc
